## 🎈프로젝트 소개

* 개요

보험사 RFP 문서를 분석하여 비기능 요구사항을 도출한다. <br/>
비기능 요구사항은 그에 맞는 QAS 시나리오를 갖는다. <br/>
품질 향상을 위해 아키텍처를 설계한다. <br/>

* 진행 기간
2021.09 ~ 2021.12

**Front-end**
